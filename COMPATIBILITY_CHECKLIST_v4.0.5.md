# Compatibility Checklist v4.0.5

## Android

1. Abrir pelo Chrome ou Edge.
2. Confirmar selo `v4.0.5` visível.
3. Abrir a aba **Compatibilidade**.
4. Executar **Auditar navegador**.
5. Testar **Instalar app** ou **Adicionar à tela inicial**.
6. Reabrir como PWA e confirmar se o status muda para instalado.

## iPhone / iPad

1. Abrir pelo Safari.
2. Usar **Compartilhar > Adicionar à Tela de Início**.
3. Reabrir pelo ícone instalado.
4. Executar auditoria na aba **Compatibilidade**.
5. Confirmar safe area/notch, rolagem, dock e botões.

## Desktop / Tablet

1. Testar Chrome/Edge.
2. Validar navegação por abas.
3. Executar QA Final e Compatibilidade.
4. Exportar relatório JSON.

## Cache/PWA

1. Executar **Limpar caches antigos**.
2. Fechar e reabrir o app.
3. Confirmar build `20260622-193123`.
