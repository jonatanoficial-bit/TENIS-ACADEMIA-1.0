# Help Center v4.0.9

Esta build adiciona uma central de ajuda offline para teste público e homologação mobile.

## Guias incluídos
- Primeiros passos.
- Gestão avançada.
- Mobile/PWA.

## Exportação
A central gera um JSON local com versão, build, schema, checklist e notas da release. Não há envio automático de dados para servidor.

## Teste recomendado
1. Abrir Dashboard.
2. Abrir Ajuda/Release.
3. Marcar guia inicial como visto.
4. Exportar relatório de suporte.
5. Testar em Android Chrome e iOS Safari.
