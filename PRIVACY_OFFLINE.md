# Privacidade e dados — Vale Games Tennis Manager v4.0.0

Esta build foi projetada como jogo offline/PWA sem backend obrigatório.

## Dados salvos

O jogo usa armazenamento local do navegador para salvar:

- carreira;
- elenco;
- rankings;
- economia;
- torneios;
- progresso de carreira;
- configurações de interface.

## Coleta externa

Esta build não implementa venda de dados, anúncios, rastreamento externo ou envio obrigatório para servidor.

## Observação

Se no futuro o jogo receber login, placar online, ranking global ou analytics, será necessário atualizar esta política antes da publicação.
