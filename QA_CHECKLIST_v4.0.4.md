# QA Checklist v4.0.4

## Teste manual recomendado no celular
1. Abrir o jogo no GitHub Pages.
2. Confirmar selo `v4.0.4` visível.
3. Entrar em QA Final.
4. Rodar **Smoke test final**.
5. Rodar **Verificar telas**.
6. Gerar relatório QA.
7. Ativar modo teste público.
8. Navegar por Partida, Atletas, Calendário, Economia, Legado e Performance.
9. Fechar e reabrir o PWA para validar cache e save.

## Critérios de aprovação
- QA Score >= 90.
- Save local funcionando.
- Sem tela branca.
- Dock mobile com rolagem.
- Partida abrindo sem erro.
- Assets com fallback se a internet oscilar.
